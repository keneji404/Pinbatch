# Pinbatch

A pinterest batch downloader web extension for Firefox.

---

### Features

- Download all pins from a board.
- Automatically fetches the highest quality of an image.

---

### Usage Guide

1. Download the files.
2. Go to `about:debugging#/runtime/this-firefox` in your browser.
3. Click **Load Temporary Add-on** and pick `manifest.json`.
4. Go to your **Pinterest Profile Page** and pick a board.
5. Start Fetching and Downloading your pins.

---

### WiP Features

- Full Automation (no more manually scrolling or zooming out to fetch all pins).
- Adding Archive options for downloads `.zip .rar`.
- Install the extension using xml from Firefox (no more debug mode).

---

### Dev Notes

I made this small project to help me batch download some of the art references I saved in Pinterest since the extensions I found are not available on Firefox or locked behind a paywall, although not perfect, it gets the job done.
